# Gammal Tech UX Design

This project presents a mobile UX design for a connected healthcare ecosystem.

## Screens Included

* Home / Health Score
* Meal Log
* AI Warning
* Doctor Booking
* Profile

## Concept

The design focuses on connecting user lifestyle data (meals, activity) with AI-driven insights and medical services.

Users can:

* Track meals and calories
* Monitor their health score
* Receive AI-based health warnings
* Book doctors based on their condition

## Key Idea

The system creates a continuous feedback loop:
User behavior → Data → AI analysis → Action (doctor / lifestyle change)
